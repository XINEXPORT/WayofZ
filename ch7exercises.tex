\section{Exercise 7.8.1}
Define the total operations T\_Backward and T\_Delete. You may use any function or operators defined in Appendix D.

\section{Exercise 7.8.2}
Define the operation that handles input characters that are not handled by Insert, Forward, Backward, and Delete.

\section{Exercise 7.8.3}
Our editor would be far more useful it it could read in the contents of a file at the user's demand. Model this in Z.

\section{Exercise 7.8.4}
How many distinct states are described by the Editor schema? The EOF schema? Assume that CHAR is implemented by the ASCII character set.

\section{Exercise 7.8.5}
Prototype the Insert operation in a functional programming language. Investigate the performance of your prototype on sequences of different sizes. (Hint: The Z operators head, tail, and \cat resemble the Lisp functions car, cdr, and append.)
